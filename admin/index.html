¿Cómo administrar el sitio?

Cualquier cambio que realices en este panel (agregar un producto, cambiar el número de WhatsApp, modificar una receta) se guardará de inmediato y actualizará el archivo js/data.js del sitio en tiempo real.

💡 Recordá: Si usás Netlify, Vercel o Github Pages, podés copiar o subir los archivos del sitio actualizados una vez guardados tus cambios.
